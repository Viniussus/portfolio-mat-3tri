Progressão Aritmética (PA):
Uma Progressão Aritmética é uma sequência numérica em que a diferença entre termos consecutivos é constante. Essa diferença constante é chamada de razão, representada por "r". A fórmula geral para um termo qualquer an de uma PA é dada por:
an = a1 + (n - 1)r
Onde: an é o n-ésimo termo, a1 é o primeiro termo, n é a posição do termo na sequência, r é a razão da PA.

Principais Características:
Crescimento Linear: Uma PA exibe um crescimento linear, pois a diferença entre os termos é constante. Se a razão r for positiva, a sequência aumenta à medida que avança.
Decrescimento Linear: Se a razão r for negativa, a PA apresenta um decrescimento linear. A sequência diminui à medida que avança.
Soma dos Termos (Soma de uma PA): A soma dos n primeiros termos de uma PA, denotada por Sn, pode ser calculada pela fórmula:
Sn = ((a1 + an)n)/2

Progressão Geométrica (PG):
Uma Progressão Geométrica é uma sequência numérica em que o quociente entre termos consecutivos é constante. Esse quociente constante é chamado de razão, representada por "q". A fórmula geral para um termo qualquer an de uma PG é dada por: an= a1 * q(n-1)
Onde: an é o n-ésimo termo, a1 é o primeiro termo, n é a posição do termo na sequência, q é a razão da PG.

Principais Características:
Crescimento Exponencial: Uma PG exibe crescimento exponencial, pois cada termo é obtido multiplicando o anterior pela razão q.
Decrescimento Exponencial: Se a razão q estiver entre 0 e 1, a PG exibe um decrescimento exponencial à medida que avança.
Soma dos Termos (Soma de uma PG finita): A soma dos n primeiros termos de uma PG finita, denotada por Sn, pode ser calculada pela fórmula:
Sn = a1 * ((1 - qn)/(1 - q))
Soma dos termos de uma PG infinita: Pode ser calculada pela fórmula:
Sn = a1 / (1 - q) para -1 < q < 1
